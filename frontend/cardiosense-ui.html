Project Documentation
About CardioSense
🎯 Project Objective

Build a binary classification model to predict heart disease risk using the Heart Disease UCI Dataset. The system trains on 13 clinical features and outputs a probability-based risk assessment.

📊 Dataset

Heart Disease UCI Dataset from Kaggle/UCI Repository. 303 patient records, 13 features, 1 binary target (0 = no disease, 1 = disease). Missing values filled with median.

🤖 Models Used
  • Logistic Regression — Features scaled with StandardScaler. L2 regularization.
  • Decision Tree — Max depth 5 to prevent overfitting. No scaling needed.
📐 Evaluation Metrics
  • Accuracy — % correct predictions
  • ROC-AUC — Area under ROC curve (discrimination ability)
  • Confusion Matrix — TP, TN, FP, FN breakdown
  • Feature Importance — DT Gini importance / LR coefficients

🚀 How to Run Locally
# 1. Clone repository
git clone https://github.com/yourusername/heart-disease-predictor
cd heart-disease-predictor

# 2. Install Python dependencies
pip install -r backend/requirements.txt

# 3. Download dataset from Kaggle → save as data/heart.csv

# 4. Train the model
cd backend && python model.py

# 5. Start API server
uvicorn app:app --reload

# 6. Open frontend/index.html in browser
